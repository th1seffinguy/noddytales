# NoddyTales Color Palette

| Name        | Hex       | CSS Var         | Role                                      |
|-------------|-----------|-----------------|-------------------------------------------|
| Sage Green  | #A8D5A2   | --sage          | App background, word screen               |
| Sage Dark   | #7CB876   | --sage-dark     | Unused — reserved for hover states        |
| Soft Orange | #F4A261   | --orange        | Primary cards, CTAs, Speak pill           |
| Orange Dark | #E08A45   | --orange-dark   | Button shadow, story text highlights      |
| Cream       | #FFF8EE   | --cream         | Story screen background                   |
| Cream Deep  | #FDEFD4   | --cream-deep    | Paper inset border, subtle fills          |
| Warm Yellow | #FFD166   | --yellow        | Age selection, tap states, Play Again     |
| Yellow Dark | #F2B838   | --yellow-dark   | Yellow button shadow                      |
| Ink         | #3D3528   | --ink           | Primary text                              |
| Ink Soft    | #6B5E48   | --ink-soft      | Secondary text, hints, captions           |

## Typography

- **Display / headings / buttons:** Fredoka (wght 500, 600, 700) — Google Fonts
- **Body / lede / story text:** Nunito (wght 600, 700, 800, 900) — Google Fonts
- Minimum story font size: 22px
- Button labels: Fredoka 22px bold, readable at arm's length on tablet
