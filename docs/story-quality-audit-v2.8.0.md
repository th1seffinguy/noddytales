# Story Quality Audit — v2.8.0

Reproducible audit pack. Generated by `scripts/audit-stories.js`.
Ages 2/4/6/8/10/12 × 5 stories each = 30 stories total.
Ages 6+ route through v3 (random blueprint per story). Picks vary deterministically per sample index.

Scoring rubric (1-5):
- **Humor** — does this make a kid laugh?
- **Substance** — is there a real arc (setup → problem → action → payoff)?
- **Kid agency** — is the kid the driver, not the witness?
- **Choice integration** — are chosen words load-bearing or decoration?
- **Rereadability** — would the kid ask for "again"?
- **Age fit** — vocabulary and complexity right for the age?
