# Story Quality Audit — v2.7.0 baseline

Reproducible audit pack. Generated by `scripts/audit-stories.js`.
Ages 2/4/6/8/10/12 × 5 stories each = 30 stories total.
Ages 6+ route through v3 (random blueprint per story). Picks vary deterministically per sample index.

Scoring rubric (1-5):
- **Humor** — does this make a kid laugh?
- **Substance** — is there a real arc (setup → problem → action → payoff)?
- **Kid agency** — is the kid the driver, not the witness?
- **Choice integration** — are chosen words load-bearing or decoration?
- **Rereadability** — would the kid ask for "again"?
- **Age fit** — vocabulary and complexity right for the age?

## Pre-change audit findings (v2.6.3 baseline)

Aggregate scoring is rough (1-5 each, eyeballed across 30 stories):

| Tier | Humor | Substance | Kid agency | Choice integration | Rereadability | Age fit |
|---|---|---|---|---|---|---|
| Tot (ages 2-3) | 3 | 3 | 3 | 3 | 3 | 4 |
| Little (age 4) | 3 | 3 | 3 | 3 | 3 | 4 |
| Kid (age 6) — v3 | 3 | 3 | 3 | 3 | 3 | 4 |
| Big (ages 8-10) — v3 | 3 | 3 | 3 | 3 | 3 | 4 |
| Tween (age 12) — v3 | 2 | 3 | 3 | 3 | 2 | 3 |

**Common root causes identified:**

1. **`goal_spine_v3` has no concrete goal.** Stories say *"today was the day. No matter what."* but never state what the goal IS. P5 says "Cole won" without telling the reader what was won. This is the single biggest substance gap. Fix: thread the v2 `goal` slot into v3 and reference it in goal_spine_v3 beats.

2. **`show_wrong_v3` prop is random + forgettable.** The prop comes from a random `V2_WORDS.objects` pick the user never made. "Library card broke in half" doesn't visualize. Fix: filter prop picks to breakable/visual items (umbrella/trophy/cape/balloon/wind-up toy), and write disaster beats with prop-specific imagery.

3. **`rule_loophole_v3` tool is also random.** Same issue — "Cole wobbled sideways while holding the tiny trophy." Needs prop-aware variants OR user-pick threading.

4. **`lost_snack_v3` reveal is repetitive.** "Single crumb of food on its face" / "looked away politely" fires nearly every story. Add 3+ varied reveal beats.

5. **Move integration is one-pattern.** "Cole [move] over to the visitor" / "Cole [move] around the room" — every move beat uses the same blocking. Vary the spatial framing (across the table, through the doorway, past the line, etc.).

6. **Color is decoration, never a clue.** "A faint X tint hung over the scene" is the only color beat. Per spec, color should be a visible clue / stain / glow / sign. Add color-driven beats.

7. **Mood is shoehorned.** "Cole felt suspiciously polite about this development" reads forced. Need mood-as-approach (kid acts ON the mood) not mood-as-label.

8. **Endings are nearly identical.** Two variants fire 90% of bedtime stories. Add 4+ ending variants with visual callbacks to the day's events.

9. **Tween (age 12) under-served.** Most tween stories use the same beats as big tier with minor wording. Need more tween-voice variants — particularly for show_wrong and rule_loophole.

10. **freeword2 callback unreliable.** Sometimes lands ("Later someone would swear they heard 'BOINGO' in the rafters"), sometimes doesn't. The flavor-callback safety net papers over a content gap rather than authored beats integrating it naturally.

**Top fix priorities (informing Phase 2):**

| Priority | Issue | Action |
|---|---|---|
| HIGH | goal_spine_v3 has vague goal | Wire `goal` slot through v3; author 4 new goal-explicit beats |
| HIGH | show_wrong_v3 prop is random | Bias prop pick to vivid breakables; author 3-4 prop-specific disaster beats |
| HIGH | rule_loophole_v3 tool is random | Same treatment as show_wrong prop |
| HIGH | Move-pattern monotony | Add 4-6 varied move-integration beats per blueprint |
| MEDIUM | lost_snack_v3 reveal repeats | Add 3 new culprit reveal variants |
| MEDIUM | Color is never a clue | Author 3 color-as-clue beats (color crumb, color stain, color tint that reveals) |
| MEDIUM | Mood is shoehorned | Author mood-as-approach beats (kid acts ON the mood) |
| LOW | Endings repeat | Add 4 ending variants (bedtime + anytime) |
| LOW | Tween under-served | Add 3-4 tween-only beats per blueprint |
| LOW | freeword2 inconsistent | Author 2-3 endings where freeword2 lands as a real callback |
